# 2018GGJ-Team5 鳥鳥傳紙條
* 使用滑鼠點擊操作
* 點擊目前拿信的人的前後左右可以傳送紙條
* 傳到最後的白文鳥的座位時勝利
* 傳到鸚鵡則失敗，鸚鵡會跟老師打小報告
* 要趁老師還沒轉頭時候傳紙條跟旁邊的人
* GGJ 主題 ： Transmission
